# 控制工程速查表

## PID 调参口诀

```
调 P — 从小到大，刚振荡就停
加 D — 抑制超调，让系统"有阻尼"
最后 I — 消除静差，别加太多（会振荡）

P 太大 → 振荡发散
I 太大 → 超调 + 慢速回稳 + 积分饱和
D 太大 → 控制信号抖动（对噪声敏感）
```

### 典型参数范围

| 被控对象 | Kp | Ki | Kd |
|---------|-----|-----|-----|
| 温度 (慢) | 2-10 | 0.5-2 | 0 |
| 电机速度 | 0.1-1 | 5-50 | 0-0.01 |
| 电机位置 | 10-100 | 1-20 | 0.5-5 |
| 四旋翼姿态 | 3-8 | 0.01-0.1 | 0.1-0.5 |

## LQR 调参

```
Q = diag([位置权重, 速度权重])
R = 控制代价

Q 大 → K 大 → 响应快、费电
R 大 → K 小 → 省电、响应慢
Q/R 比例 → 性能 vs 节能的平衡点

调参技巧:
1. 先设 Q=diag([1,1]), R=1 (起点)
2. 想更快 → Q(1) × 10
3. 想省力 → R × 10
4. 重复 2-3 直到满意
```

## Kalman 滤波

```
Q (过程噪声) → "我不信模型" → Q 大 → 更信测量
R (测量噪声) → "我不信传感器" → R 大 → 更信模型

工程技巧：从静止数据算 R（传感器噪声方差），再调 Q
```

## 滑模控制 SMC

```
λ: 滑模面斜率 s = λe + ė
   大的 λ → 收敛快（滑模面上）
K: 切换增益
   大的 K → 鲁棒性强但不稳定（抖振大）
φ: 边界层厚度
   大的 φ → 平滑（精度降）→ 小的 φ → 精确（抖振强）

起步参数: λ=10, K=3, φ=0.1 → 根据效果调整
```

## 互补滤波

```
α: 对高频信号的信任度 (0~1)

IMU 典型值: α = 0.98
  陀螺仪(高频) → 积分后短期准 → α
  加速度计(低频) → 长期无漂移 → (1-α)
  
α 越大 → 响应快但漂移大
α 越小 → 稳定但响应慢
```

## 常用 MATLAB 命令

```matlab
% 传递函数
G = tf([num],[den])

% 状态空间
sys = ss(A,B,C,D)

% 离散化
sys_d = c2d(sys, Ts, 'zoh')   % ZOH
sys_d = c2d(sys, Ts, 'tustin') % Tustin

% LQR
[K,S,e] = lqr(A,B,Q,R)

% Kalman (连续)
[~,L,P] = lqe(A,eye(n),C,Q_proc,R_meas)

% 阶跃响应
[y,t] = step(sys)
stepinfo(y,t)  % 提取性能指标

% Bode 图
bode(sys)      % 幅频 + 相频
margin(sys)    % 带裕度标注

% 根轨迹
rlocus(sys)
sgrid(zeta,wn) % 叠加设计约束
```
